ID: BUG-001
Заголовок: Метод contains работает с подстроками, хотя документация указывает "символ"
Приоритет: Low
Компонент: contains
--------------------------------------------------------------------------------
Описание:
В документации метода contains указано, что метод ищет "символ", но реализация 
позволяет искать произвольные подстроки. Это вводит пользователей в заблуждение.

Шаги воспроизведения:
1. Вызвать StringUtils().contains("SkyPro", "kyP")

Ожидаемый результат (согласно документации):
- Для "kyP" → False (так как это не один символ)

Фактический результат:
- "kyP" → True (подстрока найдена)
--------------------------------------------------------------------------------

ID: BUG-002
Заголовок: Метод capitalize не принимает пробелы в начале строки
Приоритет: Medium
Компонент: capitalize
--------------------------------------------------------------------------------
Описание:
Метод capitalize не принимает начальные пробелы, хотя документация не описывает
такое поведение. Пользователь ожидает, что изменится только регистр первой буквы.

Шаги воспроизведения:
StringUtils().capitalize("  hello world")

Ожидаемый результат:
"  Hello world" (пробелы сохраняются)

Фактический результат:
AssertionError
